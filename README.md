# General-Knowledge-Quiz-Game
A simple Python quiz game that asks users general knowledge questions, checks answers using if-else conditions, calculates the score, and displays the final result. This project demonstrates the use of variables, user input, conditional statements, and basic program flow in Python.
🧠 General Knowledge Quiz Game

A simple Python-based quiz game that tests users with general knowledge questions. The program asks multiple questions, checks the answers using conditional statements, calculates the score, and displays the final result.

🚀 Features
Interactive quiz with 3 questions
Accepts user input
Checks answers using if-else statements
Calculates and displays the final score
Provides feedback based on performance
